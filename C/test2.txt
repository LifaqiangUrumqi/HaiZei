有以下程序





#include <stdio.h>



int main() {

    int c = 35;

    printf("%d\n", c & c);

    return 0;

}


程序运行后的输出结果是 (      ) 


A.
0

B.
70

C.
35

D.
1

正确答案是 C， 回答正确  收藏 
&是按位与操作符
二进制与运算规则:1&1=1  1&0=0  0&0=0
如此题：
35化为2进制为100011
这个与操作是一位一位的操作
c&c表示
100011和100011的每一位分别进行与操作
等于100011
结果仍是35
所以输出35
